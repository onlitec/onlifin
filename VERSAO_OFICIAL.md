# Versão Oficial Onlifin
## Identificador: v1.22.0
## Data de Congelamento: 2026-03-04

Esta versão é considerada a **Versão Oficial Estável** da aplicação Onlifin.
Ela contém as seguintes implementações aprovadas:

1. **Layout de Contas Premium**: Grid moderno, dashboard de ativos e cards glassmorphism.
2. **Sistema de Familiares (Ocultar Titular)**: Opção de esconder o perfil padrão via Configurações.
3. **UI de Transações Refinada**: 
   - Contornos coloridos (Verde/Vermelho) para identificação rápida.
   - Barras de destaque lateral por tipo de fluxo.
   - Quebra automática de texto para descrições longas (Descritivos Bancários).

4. **Gerenciamento Flexível de Membros**:
   - O membro "Principal (Geral)" agora pode ser excluído se houver outros membros cadastrados.
   - Promoção automática de um novo membro principal ao excluir o antigo.

**AVISO**: Não alterar componentes visuais de Transactions ou Accounts sem consulta prévia, para manter a consistência desta versão oficial.
